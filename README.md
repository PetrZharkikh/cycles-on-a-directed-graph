# Loop Nesting Tree (Havlak)

## Описание

Проект реализует построение дерева циклов для ориентированного графа потока управления с узлами `start` и `stop`.

В основе лежит алгоритм `analyze loops` из статьи:

- Paul Havlak, *Nesting of Reducible and Irreducible Loops*, ACM TOPLAS, 1997.

Реализация определяет:

- множества узлов для loop blocks;
- заголовок (`header`) каждого цикла;
- тип цикла (`reducible` / `irreducible`);
- дерево вложенности циклов.

## Формат входа

Ввод читается из `stdin` в формате из задания:

`{start,1}, {1,2}, {2,3}, {3,stop}`

Допускаются пробелы и переносы строк.

## Формат выхода

Программа печатает:

- `Loop blocks` — разбиение узлов по самым внутренним циклам (включая корневой цикл `0`);
- `header` и тип для каждого цикла (`id >= 1`);
- `Loop tree` — пары `{parent, child}`.

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Запуск

```bash
printf "{start,1}, {1,2}, {1,3}, {2,3}, {3,4}, {4,3}, {4,5}, {5,2}, {5,stop}\n" | ./build/loop_tree
```

или интерактивно:

```bash
./build/loop_tree
```

и далее ввести список ребер в формате `{u,v}`.

## Структура проекта

- `include/LoopFinder.h` — интерфейс и структуры данных;
- `src/LoopFinder.cpp` — реализация алгоритма;
- `src/main.cpp` — чтение входа и запуск анализа.
